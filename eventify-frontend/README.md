# Eventify - Frontend (React + Vite + Tailwind)

Ceci est une version front-end minimale pour le projet **Eventify** basée sur le cahier des charges que vous avez fourni:  
Fichier source du cahier des charges: `/mnt/data/Cahier des charges - Eventify.pdf` (fourni par l'utilisateur). fileciteturn0file0

## Contenu
- Pages: `/` (Home), `/login`, `/register`, `/event/:id`, `/dashboard`, `/my-events`
- React Router pour la navigation
- Tailwind CSS (configuration incluse)

## Installation (édition dans VSCode)
1. Ouvrir le dossier `eventify-frontend` dans VSCode.
2. Installer les dépendances:
   ```bash
   npm install
   ```
3. Lancer le serveur de développement:
   ```bash
   npm run dev
   ```

## Remarques
- Backend non inclus — les formulaires utilisent des valeurs statiques/mock.
- Vous pouvez connecter une API (Node/Express, Django, Firebase) ensuite.
- N'hésitez pas à me demander d'ajouter des composants (formulaire de création d'événement, export CSV, authentification JWT...) — je peux générer les pages ou composants supplémentaires.
